7. Root-Cause Debugging via Live Testing

  • First screenshot came back completely unstyled
  • Diagnosis: postcss.config.js was missing entirely — Tailwind had never been running, in the whole app's history
  • A defect invisible to code review, only found by actually running the app
frontend/postcss.config.js (new)

  • Confirmed the new header, footer, and dashboard render correctly post-auth, with zero console errors
